구글 드라이브의 HTML · Markdown을 이쁘게

드라이브에 있는 HTML · Markdown 파일을 웹페이지 형태로 완벽하게 봅니다. 두 가지 방법 중 하나로 파일을 여세요.

  1. 구글 로그인 → 드라이브에서 열기 — 내 비공개 파일도 공개하지 않고 선택해서 봅니다. (내가 고른 파일에만 접근하는 drive.file 권한)
  2. 공유 링크 붙여넣기 — ‘링크가 있는 모든 사용자’로 공유된 파일을 상단 입력창에 붙여넣고 열기.
